# Enterprise-Network-Design

This project is a Cisco Packet Tracer network topology and configuration lab designed to simulate an enterprise network environment.

## Features
- VLAN configuration
- Inter-VLAN routing
- OSPF dynamic routing
- DHCP Relay configuration
- Centralized DHCP server
- SSH remote access
- Port Security implementation
- Subnetting (/24 divided into four /29 subnets)
- Switch and router configuration

## Technologies Used
- Cisco Packet Tracer
- Cisco IOS CLI
- Routing & Switching

## Network Services
- DHCP address distribution from centralized server
- Secure remote management using SSH
- Dynamic routing with OSPF
- Layer 2 security with Port Security
- Custom subnetting and IP addressing

## Challenges Faced
During the implementation of this lab, several networking issues were identified and resolved, including:
- Incorrect VLAN assignments and trunk configuration issues
- OSPF neighbor adjacency problems
- DHCP relay communication failures
- IP subnetting and addressing conflicts
- SSH remote access configuration errors
- Port Security violations and interface shutdown issues
- Routing inconsistencies between VLANs

## Purpose
The purpose of this lab is to practice enterprise-level networking concepts and improve routing, switching, subnetting, and network security configuration skills.
